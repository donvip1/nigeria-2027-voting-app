## Submit To Google

1. Go to:

```text
https://search.google.com/search-console
```

2. Add this property:

```text
https://nigeria-2027-voting-app.vercel.app/
```

3. Verify ownership using the option Google gives you.
4. Submit this sitemap:

```text
https://nigeria-2027-voting-app.vercel.app/sitemap.xml
```

## Important Note

SEO indexing is not instant. Google may take days or weeks to discover and rank the app. Better ranking will depend on original content, trusted backlinks, clear page purpose, and avoiding misleading official-election claims.
